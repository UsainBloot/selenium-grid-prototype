# Selenium Grid Prototype

## Install
```
npm install
```

## Run Sequential Tests Using Selenium
```
npm run test:e2e
```

## Run Parallel Tests Using Selenium Grid
```
npm run test:grid
```

## Results

| Selenium Grid | Nodes | Browsers Avail | Duration (s) |
| ------------- |:-----:|:--------------:| ------------:|
| No            |   1   |       1        |        ~ 300 |
| Yes           |   2   |       5        |         ~ 54 |
